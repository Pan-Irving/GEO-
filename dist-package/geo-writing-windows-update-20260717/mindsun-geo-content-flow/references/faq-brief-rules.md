# FAQ · 规划表模板（GEO 标准）

> 用途：替代支柱文那种"一题一份完整 Brief"。FAQ 用 **集合级 Brief + 逐条答案要点** 两层轻规划，既不过度，又锁住被引用的关键要素。
> 配套：与《FAQ-标题规划规则》搭配使用——标题规则管"问题怎么问对"，本模板管"整套怎么规划、每条答什么"。

---

## A. 集合级 Brief（一套 FAQ 共用一份）

| 字段 | 填写内容 |
|---|---|
| **项目 / 客户** | |
| **核心实体** | （这套 FAQ 要守住/植入的主体，如品牌、产品、品类名） |
| **目标关键词** | （拆解成问题前的原始词，列出） |
| **覆盖意图维度** | 是什么 / 怎么选 / 怎么用 / 安全与禁忌 / 价格与渠道 / 与竞品对比 / 常见误区（勾选并补充） |
| **实体植入策略** | （每条答案如何自然带出核心实体；哪些问题用于守竞品对比） |
| **分组结构** | （把问题按主题/场景分组，便于成页与查漏） |
| **去重原则** | （确保每条意图唯一，列出易混淆的近义问题做合并） |
| **形式分配** | 嵌入式模块 / 独立 FAQ 合集（各放哪些） |
| **答案统一调性** | （先结论后展开、字数区间、是否带数据/来源） |

---

## B. 逐条答案要点表（每条一行 mini-brief）

> 不写整篇 Brief，只写下面几列。重点是 **C 列"核心结论"**——这一句就是 AI 会抽取引用的内容。

| # | 问题（标题） | 核心结论（一句话先给答案） | 必含要点 / 事实 / 卖点 | 植入实体 | 形式 | 字数 | 证据 / 来源 |
|---|---|---|---|---|---|---|---|
| 1 | | | | | 嵌入/独立 | | |
| 2 | | | | | 嵌入/独立 | | |
| 3 | | | | | 嵌入/独立 | | |
| … | | | | | | | |

**列说明：**
- **问题**：用提问语气、一题一意图、带核心实体（见标题规则）。
- **核心结论**：第一句直接给答案，不铺垫。AI 抽取的就是这句，决定能否被引用。
- **必含要点**：答案必须覆盖的事实/卖点，防止写散、漏掉关键信息。
- **植入实体**：这条答案要自然带出哪个实体（不硬塞）。
- **形式**：嵌入支柱文模块 / 独立 FAQ 合集。
- **字数**：FAQ 答案一般 50–150 字，控制篇幅。
- **证据/来源**：需要权威背书的（如安全、数据类）标注来源。

## C. 填写示例（示意，便于理解结构）

**集合级 Brief（节选）**
- 核心实体：XX 医用冷敷贴
- 覆盖意图：是什么 / 怎么用 / 安全禁忌 / 与面膜对比 / 价格渠道
- 实体植入策略：安全类与用法类答案中自然带出品牌；对比类用于守"和面膜区别"的语义

**逐条答案要点（节选）**

| # | 问题 | 核心结论 | 必含要点 | 植入实体 | 形式 | 字数 |
|---|---|---|---|---|---|---|
| 1 | 医用冷敷贴和面膜有什么区别？ | 医用冷敷贴属医疗器械、主打舒缓修复，面膜是化妆品、主打护理，二者定位不同。 | 资质类别差异、适用场景、成分逻辑 | XX 冷敷贴 | 独立 | 120 |
| 2 | 医用冷敷贴能天天用吗？ | 可以，但建议根据皮肤状态调整频次，敏感期可每天、稳定期适当减少。 | 频次建议、敏感期用法、注意事项 | XX 冷敷贴 | 嵌入 | 90 |

## D. 使用流程

1. 先填 **A. 集合级 Brief** —— 定整套的实体、意图覆盖、分组、去重、形式分配。
2. 再填 **B. 逐条答案要点表** —— 逐条锁结论与要点。
3. 用《FAQ-标题规划规则》的**自检清单**过一遍。
4. 长项目中，按监测到的"问题空缺"在 B 表持续增补新行（核心不动）。

---

## 一句话总结

> FAQ 不需要支柱文式完整 Brief。
> **集合级 Brief 管整套（实体/意图/去重/形式），逐条答案要点管每条（结论+要点+实体）**，
> 其中"核心结论"那一句最关键——它就是被 AI 引用的内容。
